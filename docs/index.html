Remotes

Checking Remotes

git remote -v

Adding a Remote

In this example we are adding a 'staging' remote

git remote add staging https://git.heroku.com/app-name.git

Removing a Remote

In this example we are removing the 'staging' remote

git remote rm staging

Renaming a Remote

In this example we are renaming the 'origin' remote to 'production'

git remote rename origin production

Setting push url for Remote

git remote set-url --push origin "NO PUSHING ALLOWED"

Setting the remote tracking branch

In this example we are setting the 'production' remote for tracking

git config branch.master.remote production
