### Requirements

* [NodeJs](https://nodejs.org/)
* [Yarn](https://yarnpkg.com/)

##### Install Yarn if you haven't already
```
npm install yarn -g
```

##### Install Node Modules
```
yarn
```

##### Watch
```
yarn watch
```

##### Build
```
yarn build
```

### For new projects
```
git remote rename origin upstream
```
```
git remote set-url --push upstream ""
```
